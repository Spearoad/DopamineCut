# 숏폼 역할 담당하신분들 읽어주세용

## 일정 안내
**기한**: ~ 2026. 04. 01 (수)
**목표**: 플랫폼별 숏폼 인식 및 카운팅 로직 구현 완료

---

## 주요 개발 파일 설명

### 1. UI 및 컨트롤러
- ShortformSettingActivity.kt : 숏폼 차단 설정 ui 컨트롤러와 플랫폼별 목표 사용 시간 & 목표 숏폼 시청 수를 지정
- activity_shortform_setting.xml: 숏폼 차단 설정 레이아웃

### 2. 플랫폼별 로직
YoutubeManager.kt, InstagramManager.kt, TikTokManager.kt, KakaotalkManager.kt
매니저 클래스에서는 앱 실행 확인, 쇼츠 인식, 쇼츠 카운팅, 앱 실행 시간 기록, 날짜 기록을 담당합니다.

### 3. 로직 동작 흐름
1) 대상 앱 실행 여부 확인
2) 숏폼 진입 판별
3) 숏폼 카운팅
	- 유효 시청: 영상 진입 후 일정 시간(ex: 3~5초) 머무를 시 시청으로 카운트되도록 하기
	- 중복 카운팅 방지: 사용자가 위로 올려서 동일한 영상을 시청해도 카운팅되지않도록 방지
4) 데이터 기록
	시청 영상 수, 사용 시간대, 앱 사용 시간을 앱(플랫폼) 종료 시점에 최종 기록.

### 4. 알림
설정 값 초과 시, 즉시 차단할 지 아니면 알림 및 기록만 진행할 지 유저 선택 가능
1) 초과량 기록
 - 설정한 횟수 / 시간 모두 얼마나 초과했는지 3. 4)와 분리된 추가적인 기록
3) 실시간 알림
 - (목표 80% 달성 시): "오늘 목표치까지 단 2개 남았습니다. 여기서 멈추면 당신은 승리자!"
서버 연동은 구현이 끝나면 제가 작업할테니 구현에 집중해 주시면 감사하겠습니다.
